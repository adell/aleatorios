Cow Say

Este é um clone simples do programa Cow Say, que existe a mais de 20 anos.

1. O que este programa faz?

Apenas imprime em tela o texto passado como parametro na chamada do program

$ cowsay "Olá mundo!" 
____________
< Olá mundo! >
 ------------
        \   ^__^
         \  (oo)\_______
            (__)\       )\
                ||----w | \
                ||     ||  s

2. Fonte

package main

import (
"fmt"
"os"
"strings"
)

var qtd_barra int

func diga(diga string) string {
qtd_barra = len(diga) + 1
return diga
}

func main() {
var fale string
if len(os.Args) > 1 {
fale = diga(os.Args[1])
} else {
fale = diga("Digite um texto para que eu fale!")
}
barra := strings.Repeat("-", qtd_barra)

fmt.Println(" ", barra)
fmt.Println(" (", fale, ")")
fmt.Println(" ", barra)
fmt.Println("        \\   ^__^")
fmt.Println("        \\_ (oo)\\_______")
fmt.Println("           (__)\\       )\\")
fmt.Println("               ||----w | \\")
fmt.Println("               ||     ||  s")
}

3. Executando

Para ver o programa funcionando, digite na mesma pasta do seguinte comando:

go run main.go

4. Compilando

Para gerar um executavél que pode ser transportado para outras maquinas, faça:

go build -o cowsay

O parametro -o é para nomear o binario, usando o texto que vem em seguida.
